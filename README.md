# README

FriendSpace

This project is based on the <a href="https://www.theodinproject.com/paths/full-stack-ruby-on-rails/courses/ruby-on-rails/lessons/rails-final-project">final project of The Odin Project's Ruby on Rails course</a>. It is a very lite version of Facebook. 

The main features of FriendSpace include:
<ul>
<li>
Users can register accounts via the Devise gem.
</li>
<li>
Users can create and interact with posts, such as leaving comments or likes. 
</li>
<li>
Users can upload photos as posts, create text-only posts, or add both.
</li>
<li>
Users can search for other users and send friend requests. If the receiving user accepts the request, the two users become friends.
</li>
<li>
Users can poke one another.
</li>
<li>
Users can check notifications including friend requests, comment notifications and poke notifications.
</li>
